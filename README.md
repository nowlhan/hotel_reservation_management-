#Dans le terminal, entrer dans le dossier backend et activer l’environnement virtuel. Lancer ensuite le serveur avec python manage.py runserver.
Dans un autre terminal, entrer dans le dossier frontend et lancer le serveur avec npm run dev et suivre le lien du serveur donné. 
Vous arriverez sur la partie Dashboard qui regroupe une vue simplifié des occupation et disponibilité
Dans le coté gauche supérieur de l’ecran, choisir « chambres » et vous verrez la liste des chambres et possède les fonctionnalités suivante : ajout, modification et suppression
Toujours dans le coté gauche supérieur de l’ecran, choisir « Réservation » et vous verrez la liste des réservation et possède les fonctionnalités suivante : ajout, modification et suppression
